## Nano
- `Ctrl O` — записать файл
- `Ctrl X` — выйти
- `Ctrl W` — найти строку
- `Ctrl \` — найти и заменить
- `Ctrl A` — начало строки
- `Ctrl E` — конец строки
- `Ctrl K` — удалить строку
